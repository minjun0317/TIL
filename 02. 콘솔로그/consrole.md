# 콘솔로그
➡️ 프로그램이 실행되는 동안 상태나 값(변수값), 오류등을 콘솔에 출력해서 확인하는 기능
```ts
let age: Int = 10

print("안녕하세요. 저는 \(age + 1)살 입니다.")
// == 안녕하세요. 저는 11살 입니다.
```

# Print
- 단순 문자열 출력

| **사용 예시**

.

```ts
print(Me)
// == Person
```

# Dump
- 인스턴스의 자세한 설명(description 프로퍼티)까지 출력
    - 실제로 만들어진 객체, 즉 설계도에서 실제 물건과 같은 관계를 문자열로 표현한 값을 출력한다


| **사용 예시**

.

.

```ts
dump(Me)
// == Person
// == minjun
// == 17
```
