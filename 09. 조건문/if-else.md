# if-else 구문
➡️ ```if```를 단독적으로, ```else if```,```else```와 함께 조합 사용 가능하다

| **사용 예시**

- if 뒤에 조건 값은 Bool타압 값만 위치해야한다, 소괄호는 선택
```ts
if 조건식 {
    조건식에 만족한다면 구문 실행
} else {
    조건식이 아니라면 else 구문 실행
}
```

| **사용 예시**

```ts
if 조건식 ganadi {
    조건식 ganadi이 만족하면 구문 출력
} else if 조건식 2 {
    조건식 ganadi 1를 만족하면 구문 출력
} else {
    둘 다 아니라면 구문 출력
}
```

# 비교 연산자 

- ```>```  왼쪽 값이 오른쪽 값보다 큼
- ```<```  왼쪽 값이 오른쪽 값보다 작음
- ```>=```  왼쪽 값이 오른쪽 값보다 크거나 같음
- ```<=```  왼쪽 값이 오른쪽 값보다 작거나 같음
- ```==```  왼쪽 값이 오른쪽 값과 같음
- ```!=```  왼쪽 값이 오른쪽 값과 다름
