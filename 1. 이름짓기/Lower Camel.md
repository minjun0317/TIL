## Lower camel case
- 첫단어는 소문자로 표기
- 값들을 표기
- 이후 단어들은 대문자로 표기
- 띄어쓰기나 언더바(_) 사용 불가로 이어쓰기로 표기

| **사용 예시**
- 함수(fuction)
- 메소드(method)
- 변수(variable)
- 상수(constant)

```ts
let userName = "minjun"
var studentAge = 11
func printMinjunProfile() {
}
```

## Upper Camel Case
- 첫단어는 대문자로 표기
- 타입(type) 이름들을 표기

| **사용 예시**
- 참조 타입(class)
- 값 타입(struct)
- 값 묶는 타입(enum)
- 기능 추가 타입(extension)
- ...
```ts
struct MinjunProfile {   
}
class MinjunManager {
}

```
