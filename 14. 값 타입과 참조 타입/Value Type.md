# 값 타입 ( Value Type )
➡️ 변수에 값을 직접 복사해서 저장함

| **예시**
- 한 쪽 값 바꿔도 다른 변수에 영향 없음 
    <details> 
    <summary>ex) struct, enum, Int, String
    </summary>
    - struct : 데이터 묶음
    - enum : 상태나 경우의 수 표현
    - Int : 정수
    - String : 문자열
    </details>

⚠️ 값 타입은 안전하고 예측 쉬움, 기본적으로 값 타입을 더 많이 사용해서 안정성을 높이려는 설계
