# 클로저란
➡️ 클로저(Closure)는 코드에서 일급 객체로 취급되는 함수의 일종으로 코드 블록을 캡처하고 전달할 수 있지만, 이름이 없고 주변의 변수와 상수를 캡처할 수 있다는 점에서 함수와 구분
클로저는 보통 함수나 메서드의 인수로 전달되어 나중에 실행될 코드 블록을 정의하는 데 사용

# 클로저 기본
| **사용 예시**

```ts
{ (매개변수 목록) -> 반환타입 in
    실행 코드
}
```
---
.

```ts
let ddung = {
    print("똥싸는 가나디 처음봐요?")
}

ddung() 
//== 똥싸는 가다니 처음봐요?
```
---

.

.

<details> 
<summary> 클로저 표현들
</summary>

- 함수의 매개변수 마지막으로 전달되는 클로저는 후행클로저 `trailing closure`

- 반환 값이 있는 경우 암시적으로 클로저의 맨 마지막 줄은 `return` 키워드를 생략하더라도 반환 값으로 취급
- 전달 인자의 이름이 굳이 필요없다
- 컴파일러가 타입을 유추할 수 있는 경우 축약된 전달인자 이름 ($0, $1, $2 ...)를 사용 가능
</details>
