 ➡️ 구조체, class 같은 곳에 타입과 연관된 값을 표현할 때 사용하며 구조체, 클래스, 열거형 내부에 구현 가능하다

.

.

# 프로퍼티의 종류

## 저장 프로퍼티
➡️ 값을 저장하기 위해 선언하는 변수 혹은 상수
- 클래스나 구조체에서만 활용 가능

.

| **사용 예시**

```ts
struct ganadi {
    var name: String = "가나디"
}

let g = ganadi()
print(g.name) 
//== 가나디
```

### 읽기 전용 ( Get ) 
- Get는 저장 프로퍼티의 값을 임의로 연산, 반환함
    - var와 명칭, 그리고 선언된 자료 뒤에 {}블럭으로 감싸주고
    - 내부에 get {}코드 블럭을 활용하여 연산 실행


### 쓰기 전용 ( Set ) 
- set는 저장 프로퍼티의 값을 간접적으로 설정, 반환함
    - 연산 프로퍼티 내부에 set{} 코드블럭을 활용
    - 해당 연산 프로퍼티의 값을 ➜ (매개변수)로 넘겨주고, 내부에서 연산을 실행
---

## 연산 프로퍼티
➡️ 연산을 실행할 수 있지만 저장은 할 수 없는 프로퍼티이다 연산 프로퍼티는 직접 값을 가질 순 없지만 다른 프로퍼티로 연산값을 부여 가능하다

| **사용 예시**

```ts
struct ganadi {
    var name: String = "가나디"
    
    var g: String {
        return "나는 똥싸는 \(name)야" 
    }
}

let ga = ganadi()
print(ga.g)
```
---

| 구분      | 어디에 선언           | 사용 범위       |
| ------- | ---------------- | ----------- |
| 타입 프로퍼티 | struct / class 안 | 모든 인스턴스가 공유 |
| 전역 변수   | 함수 밖             | 전체에서 사용     |
| 지역 변수   | 함수 안             | 그 안에서만 사용   |
