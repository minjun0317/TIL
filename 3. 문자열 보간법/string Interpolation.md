# 문자열 보간법
➡️ 프로그램 실행 중 문자열 내에 변수 또는 상수의 실질적인 값을 표현하기 위해 사용

| **사용 예시**
```
\(값) 
```
➡️ 괄호 사이에 값을 넣어주면 문자열에서 자동으로 치환해서 나타남

---

| **사용 예시**

.

```ts
let minjunAge = 11
let school = "마이스터고"

let intro = "\(minjunAge)살 \(school) 학생 민준입니다"
// == 11살 마이스터고 학생 민준입니다

let a = 1
let b = 2

print("a + b = \(a + b)")
// ==  a + b = 3
```
