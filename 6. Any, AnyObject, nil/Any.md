# Any
➡️ Swift의 모든 타입을 지칭하는 타입, 데이터 타입이 ```Any``` 선언이 되었다면 어떤 타입이든 수용 가능하다

| **사용 예시**

- Any는 다른 타입을 계속 넣어도 됨

.
```ts
var all: Any
//== 모든 타입 값을 담을 수 있는 변수

all = 1

all = 3.141592623..

all = "가나디안아줘"

all = "가나디안지마"
//== 다른 값(String) 덮어쓰기 가능

all = ["가","나","디"]

all = {
    print("가나디 티쳐스 비싸ㅜ")
}
```
#### ```Any```는 타입의 안정성을 희생
