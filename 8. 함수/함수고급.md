- 기본값을 가지는 매개변수는 목록 중에 뒤쪽에 위치하는 것이 좋다

.

```ts
func ganadi(name : String, message : String = "가나디 안아줘") -> String {
    return "(message), (name)"
}

print(ganadi(name : "안아주지마"))
print(ganadi(name : "미피는", message : "안아주자"))
//== 가나디 안아줘, 안아주지마
//== 안아주자, 미피
```

# 전달인자 레이블

➡️ 함수를 호출할 때 함수 사용자의 입장에서 값이 어떤의미 인지 알려주는 설명글이다

|보는 위치|보이는 것|의미|
|:---|:---|:---|
|함수 호출 할 때|```to:```,```from:```|값의 역할 설명|
|값 앞에 붙음|```to: "가나디"```|누구에게 보내는지 알려줌|
|코드 읽을 때|문장처럼 읽힘|이해 쉬움|

| **사용 예시**

```ts
func ganadi(to owner: String, from todo: String) {
    print("가나디를 (owner)번 부터 (todo)번 안아줘")
}

ganadi(to: 1, from: 100) 
//== 가나디를 1번 부터 100번 안아줘
```

# 가변 매개변수

➡️ 전달 받을 값이 얼마나 들어올지 알기 어려워서 여러개 받을 수 있게 만든 매개변수

| **사용 예시**
- 함수 내에서 배열처럼 사용 가능하다
    - '...' 여러개 받을 수 있다는 뜻

.

```ts
func ganadi(someganadi: Int...) -> Int {
    var total = 0
    for someganadi in points {
        total += someganadi
    }
    return total
}

print(ganadi(someganadi: 1, 2, 3))
//== 6
```
복잡하기 때문에 보기싫어져서 구조도로 나타내기
```
입력 → 1,2,3
        ↓
someganadi = [1,2,3]
        ↓
전부 더함
        ↓
6 반환
```

# 튜블 반환
➡️ 함수는 튜플을 사용해 여러 데이터를 반환할 수 있다
- 튜플 = 여러 값을 하나로, 서로 다른 타입도 가능하다

| **사용 예시**
```ts
func ganadi(a: Int, b: Int) -> (sum: Int, total: Int) { 
// 'ganadi' 라는 함수 선언, a, b 두 개의 정수 매개변수를 받고 sum : Int, total : Int) ... 튜프로 값 2개 변환
    let sum = a + b
    let total = a * b
    return (sum, total) // 합이랑 곱을 튜플로 묶어서 반환, (sum, total) ... 값 2개 동시에 리턴
}

let result = ganadi(a: 6, b: 7)
print("가나디 인형의 합 (result.sum), 가나디의 면적 (result.total)") 
// == 가나디 인형의 합 합 7, 전체 세트 12
```
