# 함수 기본
➡️ 함수는 특정한 작업을 수행하는 코드블럭이다

| **사용 예시**

- 함수의 정의 ( 함수를 사용하면 코드의 재사용성과 가독성이 증가함 ) 

.

```ts
func 함수이름(매개변수이름 : 매개변수타입) -> 반환타입 {
    함수가 수행할 작업
    return 반환값
}
```
---
```ts
func roomdecoration(furniture : String) -> String {
    let plan = "내방 가구는 (furniture)밖에 없어요"
}

let tobuy = roomdecoration(furniture : "소파")

print(tobuy)
//== 내방 가구는 소파밖에 없어요
```

# 매개변수(함수에 지정해주는 값)가 없는 함수
| **사용 예시**

```ts
func furniture() -> String {
    return 내방 가구는 소파밖에 없어요
}

let tobuy = furniture()

print(tobuy)
//== 내방 가구는 소파밖에 없어요
```

# 매개변수와 반환값이 없는 함수
| **사용 예시**
- void라는 값이 없다는 용어를 반환타입에 써줘야한다

.

```ts
func tobuy() -> Void {
    print("내방 가구는 소파밖에 없어요")
    return
}
```
---
- 함수 구현이 짧으면 한 줄로 표현한다

.

```ts
func tobuy() -> Void {print("내방 가구는 소파밖에 없어요")}
```
---
- Void 생략도 가능하다

.

```ts
func tobuy() {print("내방 가구는 소파밖에 없어요")}
```
