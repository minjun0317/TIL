# 이름짓기

## Lower camel case
-첫단어는 소문자로 표기
-이후 단어들은 대문자로 표기
-띄어쓰기나 언더바(_) 사용 불가로 이어쓰기로 표기

**사용 예시**
- 함수(fuction)
- 메소드(method)
- 변수(variable)
- 상수(constant)100


```swift
print("ㅇ리;마ㅓㄴ이ㅏ러") //fsfgsd
```

| 안녕 | 하세요 | 헬로우웅 |
|------:|:------|------:|
| 핼로우 | 짜이찌엔 | 이ㅓ링 |
| 핼로우 | 짜이찌엔 | 이러;닝러 |
| 핼로우 | 짜이찌엔 | ㄴ이ㅏ룽누 |
| 핼로우 | 짜이찌엔 | ㅇ니러ㅐㄴ;ㅓ |
